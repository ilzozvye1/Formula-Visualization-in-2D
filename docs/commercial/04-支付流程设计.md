## 二、支付方式选择

### 2.1 支付方式对比

| 方式 | 申请难度 | 手续费 | 到账时间 | 适用场景 |
|------|----------|--------|----------|----------|
| 微信收款码 | ⭐ 即时开通 | 0% | 即时 | 个人转账 |
| 支付宝收款码 | ⭐ 即时开通 | 0% | 即时 | 个人转账 |
| 微信小商店 | ⭐⭐ 1-3天 | 0.6% | 即时 | 正规经营 |
| 支付宝商家 | ⭐⭐ 1-3天 | 0.6% | 即时 | 正规经营 |
| PayPal | ⭐⭐ 3-7天 | 2.9%+$0.3 | 即时 | 海外用户 |
| Paddle | ⭐⭐⭐ 5-10天 | 5% | 3天 | 订阅制 |

### 2.2 推荐方案

#### 初期（个人开发者）

使用**微信/支付宝个人收款码**：

```
优点：
- 即时开通，无需审核
- 手续费为0%（个人转账）
- 简单直接

缺点：
- 需要人工确认支付
- 无法自动发送密钥
- 不够正规

适用阶段：产品初期验证
```

#### 正规化后

使用**微信小商店/支付宝商家**：

```
优点：
- 自动确认支付
- 可自动发送密钥
- 更正规可靠

缺点：
- 需要营业执照
- 有手续费0.6%

适用阶段：产品稳定运营
```

## 五、支付确认流程

### 5.1 人工确认模式

```javascript
// 用户提交支付凭证
const paymentConfirm = {
  // 1. 用户提交支付凭证
  async submitPaymentProof(orderData) {
    const order = {
      id: generateOrderId(),
      plan: orderData.plan,
      amount: orderData.amount,
      paymentMethod: orderData.method,
      email: orderData.email,
      status: 'pending',
      createdAt: new Date()
    };
    
    // 保存订单
    await db.orders.save(order);
    
    // 返回订单信息
    return {
      success: true,
      orderId: order.id,
      message: '请完成支付后截图发送给我们'
    };
  },
  
  // 2. 管理员确认支付
  async confirmPayment(orderId, licenseKey) {
    const order = await db.orders.find(orderId);
    if (!order) {
      return { success: false, message: '订单不存在' };
    }
    
    // 生成许可证密钥
    const key = licenseKey || LicenseKeyGenerator.generate(order.planType);
    
    // 更新订单状态
    await db.orders.update(orderId, {
      status: 'completed',
      licenseKey: key,
      completedAt: new Date()
    });
    
    // 发送密钥到邮箱
    await sendLicenseEmail(order.email, key);
    
    return { success: true, message: '支付确认，密钥已发送' };
  },
  
  // 3. 发送邮件
  async sendLicenseEmail(email, licenseKey) {
    const mailContent = `
      您好！
      
      感谢您购买公式可视化 Pro 版！
      
      ═══════════════════════════════
      您的许可证密钥：${licenseKey}
      ═══════════════════════════════
      
      激活方法：
      1. 打开公式可视化
      2. 点击"激活Pro版"
      3. 输入上方密钥
      4. 享受Pro版全部功能
      
      如有任何问题，请联系客服。
      
      公式可视化团队
    `;
    
    // 使用邮件服务发送
    await emailService.send({
      to: email,
      subject: '【公式可视化】Pro版许可证密钥',
      body: mailContent
    });
  }
};
```

### 5.2 自动化确认模式

```javascript
// 支付回调处理
async function handlePaymentCallback(paymentData) {
  const { orderId, transactionId, amount, status } = paymentData;
  
  if (status !== 'success') {
    return { success: false, message: '支付失败' };
  }
  
  // 1. 验证订单
  const order = await db.orders.find(orderId);
  if (!order) {
    return { success: false, message: '订单不存在' };
  }
  
  // 2. 验证金额
  if (order.amount !== amount) {
    return { success: false, message: '金额不匹配' };
  }
  
  // 3. 生成密钥
  const key = LicenseKeyGenerator.generate(order.planType);
  
  // 4. 更新订单
  await db.orders.update(orderId, {
    status: 'completed',
    licenseKey: key,
    transactionId,
    completedAt: new Date()
  });
  
  // 5. 自动发送密钥
  await sendLicenseEmail(order.email, key);
  
  // 6. 通知用户
  await notifyUser(order.userId, {
    type: 'payment_success',
    licenseKey: key
  });
  
  return { success: true };
}
```
