## 六、支付流程

### 6.1 支付方式

| 方式 | 手续费 | 适用场景 |
|------|--------|----------|
| 微信收款码 | 0% | 初期个人转账 |
| 支付宝收款码 | 0% | 初期个人转账 |
| 微信小商店 | 0.6% | 正规化后 |
| 支付宝商家 | 0.6% | 正规化后 |

### 6.2 流程阶段

**初期（人工模式）**：
1. 用户选择套餐
2. 显示收款码
3. 用户扫码支付
4. 用户截图发送给你
5. 你确认后发送密钥

**正规化后（自动模式）**：
1. 用户选择套餐
2. 跳转支付
3. 支付成功回调
4. 系统自动发送密钥
