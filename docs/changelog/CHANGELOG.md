# 更新日志 (Changelog)

## [1.1.0] - 2025-03-04

### 新增功能 (Added)

#### 3D可视化功能
- **3D模式切换**: 在设置按钮旁添加2D/3D切换按钮，可自由切换2D与3D坐标系
- **3D空间曲线**: 添加12种3D空间曲线预设
  - 螺旋线 (Helix)
  - 三叶结 (Trefoil Knot)
 - 环面结 (Torus Knot)
  - 利萨茹曲线 (Lissajous)
  - 维维亚尼曲线 (Viviani)
  - 球面螺旋线
  - 圆锥螺旋线
  - 3D玫瑰线
  - 扭曲立方曲线
  - 3D正弦波
  - 8字结 (Figure Eight)
  - 五叶结 (Cinquefoil)
- **3D曲面**: 添加10种3D曲面预设
  - 平面 (Plane)
  - 球面 (Sphere)
  - 圆锥面 (Cone)
  - 抛物面 (Paraboloid)
  - 单叶双曲面 (Hyperboloid)
  - 马鞍面 (Saddle)
  - 波浪面 (Wave)
  - 环面 (Torus)
  - 高斯曲面 (Gaussian)
  - 涟漪面 (Ripple)

#### 3D交互功能
- **自动旋转**: 添加自动旋转按钮，支持3D视图自动旋转展示
- **旋转速度控制**: 在设置菜单中添加旋转速度滑块（1-10档）
- **视角重置**: 添加正视图、俯视图、侧视图快速切换按钮
- **鼠标旋转**: 3D模式下支持鼠标拖拽旋转视角

#### 可视化增强
- **深度雾化**: 添加深度雾化效果，根据距离调整线条透明度
- **图例显示**: 在画布上显示当前可见方程的图例
- **公式显示**: 3D曲线和曲面显示数学公式
- **参数编辑**: 支持实时编辑3D曲线和曲面的参数

#### 数据导出
- **导出3D数据**: 支持将3D场景数据导出为JSON格式

### 改进 (Changed)
- 优化UI布局，调整图例和坐标显示位置
- 改进坐标显示样式，更加紧凑
- 支持暗色模式下的3D可视化

### 修复 (Fixed)
- 修复3D曲线在方程列表中不显示的问题
- 修复坐标显示和图例重叠的问题

## [1.0.0] - 2025-03-03

### 初始版本 (Initial Release)
- 2D公式可视化基础功能
- 支持多种方程类型（线性、二次、幂函数、指数、对数、三角函数等）
- 支持微分和积分可视化
- 支持交点检测和显示
- 支持方程拖拽调整
- 支持导入/导出方程列表
- 支持暗色模式
- 支持导出图像
