# 动画功能使用指南

## 概述

动画功能允许您为方程参数和视图创建动态效果，让数学公式"活"起来。通过动画，您可以直观地展示参数变化对函数图像的影响。

---

## 🎬 功能特性

### 1. **参数动画**
- 振幅动画（Amplitude）
- 频率动画（Frequency）
- 相位动画（Phase）
- 支持任意参数的动态调整

### 2. **视图动画**
- 缩放动画
- 平移动画
- 组合视图变换

### 3. **预设动画**
- 振幅波动
- 频率变化
- 相位旋转
- 缩放脉冲
- 视图平移

### 4. **高级功能**
- 9 种缓动函数
- 循环播放
- 往返动画（Yo-Yo）
- 多动画同时播放
- 动画状态监控

---

## 🚀 快速开始

### 步骤 1: 添加方程

首先添加一个或多个方程，例如：
```
y = sin(x)
```

### 步骤 2: 打开动画面板

点击 **"🎬 动画面板"** 按钮，展开动画控制面板。

### 步骤 3: 选择动画预设

从下拉菜单中选择预设动画：
- **振幅波动**: 振幅在 1 到 3 之间波动
- **频率变化**: 频率在 1 到 3 之间变化
- **相位旋转**: 相位从 0 到 2π 旋转
- **缩放脉冲**: 视图缩放脉冲效果
- **视图平移**: 视图平移效果

### 步骤 4: 配置动画参数

- **目标方程**: 选择要应用动画的方程
- **持续时间**: 拖动滑块设置动画时长（100ms - 5000ms）
- **缓动函数**: 选择动画的缓动效果
- **循环播放**: 勾选后动画无限循环
- **往返动画**: 勾选后动画会往返播放

### 步骤 5: 播放动画

点击 **"▶ 播放"** 按钮开始动画。

## 📋 动画预设详解

### 1. 振幅波动 (Amplitude Wave)

**效果**: 方程的振幅在一定范围内波动

**适用方程**:
- `y = A·sin(x)` - 正弦函数
- `y = A·cos(x)` - 余弦函数
- `y = A·e^x` - 指数函数

**参数**:
- 振幅范围：1 → 3
- 默认持续时间：2000ms
- 缓动函数：easeInOutSine
- 自动往返

**视觉体验**: 波形像呼吸一样扩张和收缩

---

### 2. 频率变化 (Frequency Change)

**效果**: 方程的频率在一定范围内变化

**适用方程**:
- `y = sin(f·x)` - 正弦函数
- `y = cos(f·x)` - 余弦函数
- `y = tan(f·x)` - 正切函数

**参数**:
- 频率范围：1 → 3
- 默认持续时间：3000ms
- 缓动函数：easeInOutSine
- 自动往返

**视觉体验**: 波形逐渐变密然后变疏

---

### 3. 相位旋转 (Phase Rotation)

**效果**: 方程的相位从 0 旋转到 2π

**适用方程**:
- `y = sin(x + φ)` - 正弦函数
- `y = cos(x + φ)` - 余弦函数

**参数**:
- 相位范围：0 → 2π
- 默认持续时间：4000ms
- 缓动函数：linear
- 循环播放

**视觉体验**: 波形从左向右平移，演示相位移动

---

### 4. 缩放脉冲 (Zoom Pulse)

**效果**: 视图缩放产生脉冲效果

**适用场景**:
- 所有方程类型
- 强调特定区域

**参数**:
- 缩放范围：40 → 60
- 默认持续时间：1000ms
- 缓动函数：easeInOutQuad
- 自动往返

**视觉体验**: 镜头推进然后拉回

---

### 5. 视图平移 (View Pan)

**效果**: 视图在 X 和 Y 方向平移

**适用场景**:
- 展示大范围图像
- 追踪函数行为

**参数**:
- X 偏移：0 → 100
- Y 偏移：0 → 50
- 默认持续时间：2000ms
- 缓动函数：easeInOutQuad

**视觉体验**: 镜头平滑移动

---

## 🎛️ 缓动函数详解

缓动函数控制动画的速度变化，让动画更自然。

### 线性 (Linear)
```
速度恒定，机械感
```
**适用场景**: 相位旋转、匀速运动

---

### 缓入 (Ease In)
```
慢 → 快
开始缓慢，逐渐加速
```
**适用场景**: 物体下落、加速运动

**类型**:
- `easeInQuad` - 二次方缓入
- `easeInSine` - 正弦缓入
- `easeInBounce` - 弹跳缓入

---

### 缓出 (Ease Out)
```
快 → 慢
开始快速，逐渐减速
```
**适用场景**: 物体上抛、减速运动

**类型**:
- `easeOutQuad` - 二次方缓出
- `easeOutSine` - 正弦缓出
- `easeOutBounce` - 弹跳缓出

---

### 缓入缓出 (Ease In Out)
```
慢 → 快 → 慢
开始缓慢，中间加速，结束减速
```
**适用场景**: 大多数自然运动、UI 动画

**类型**:
- `easeInOutQuad` - 二次方缓入缓出（最常用）
- `easeInOutSine` - 正弦缓入缓出（最平滑）

## 💡 使用技巧

### 1. 组合动画

同时播放多个动画创建复杂效果：

```javascript
// 示例：同时应用振幅和频率动画
1. 选择"振幅波动"，播放
2. 选择"频率变化"，播放
3. 两个动画同时作用！
```

### 2. 自定义参数动画

不使用预设，直接自定义：

```javascript
// 在控制台中执行
app.animationManager.createParameterAnimation(
    0,              // 方程索引
    'amplitude',    // 属性
    1,              // 起始值
    5,              // 结束值
    {
        duration: 2000,
        easing: 'easeInOutQuad',
        repeat: -1,  // 无限循环
        yoyo: true   // 往返
    }
);
```

### 3. 视图动画编程控制

```javascript
// 缩放动画
app.animationManager.createViewAnimation({
    fromScale: 40,
    toScale: 80,
    duration: 1500,
    easing: 'easeOutQuad'
});

// 平移动画
app.animationManager.createViewAnimation({
    fromOffsetX: 0,
    toOffsetX: 200,
    fromOffsetY: 0,
    toOffsetY: 100,
    duration: 2000
});
```

### 4. 动画控制

```javascript
// 暂停所有动画
app.animationManager.pauseAll();

// 恢复所有动画
app.animationManager.resumeAll();

// 停止特定动画
app.animationManager.removeAnimation(animationId);

// 停止所有动画
app.animationManager.stopAll();

// 获取动画状态
const stats = app.animationManager.getStats();
console.log(stats); // { total: 2, playing: 1, paused: 1 }
```

## ⚙️ API 参考

### AnimationManager 类

#### 方法

**`init(app)`**
- 初始化动画管理器
- `app`: 应用实例

**`createParameterAnimation(index, property, from, to, options)`**
- 创建参数动画
- `index`: 方程索引
- `property`: 属性名（amplitude, frequency, phase 等）
- `from`: 起始值
- `to`: 结束值
- `options`: 动画选项
  - `duration`: 持续时间（ms）
  - `easing`: 缓动函数
  - `repeat`: 重复次数（-1 为无限）
  - `yoyo`: 是否往返

**`createViewAnimation(options)`**
- 创建视图动画
- `options`: 动画选项
  - `fromScale`: 起始缩放
  - `toScale`: 结束缩放
  - `fromOffsetX`: 起始 X 偏移
  - `toOffsetX`: 结束 X 偏移
  - `fromOffsetY`: 起始 Y 偏移
  - `toOffsetY`: 结束 Y 偏移
  - `duration`: 持续时间
  - `easing`: 缓动函数

**`createPresetAnimation(name, equationIndex)`**
- 创建预设动画
- `name`: 预设名称
- `equationIndex`: 方程索引

**`pauseAll()`**
- 暂停所有动画

**`resumeAll()`**
- 恢复所有动画

**`stopAll()`**
- 停止所有动画

**`getStats()`**
- 获取动画统计信息
- 返回：`{ total, playing, paused }`

## 🐛 故障排除

### 问题 1: 动画不播放

**可能原因**:
- 没有选择方程
- 方程列表为空

**解决方案**:
1. 确保至少有一个方程
2. 在"目标方程"下拉框中选择方程

---

### 问题 2: 动画卡顿

**可能原因**:
- 方程数量太多
- 浏览器性能不足

**解决方案**:
1. 减少同时显示的方程数量
2. 关闭其他浏览器标签页
3. 使用硬件加速（在浏览器设置中开启）

---

### 问题 3: 动画无法停止

**解决方案**:
1. 点击"⏹ 停止全部"按钮
2. 刷新页面
3. 在控制台中执行：`app.animationManager.stopAll()`

## 🎯 最佳实践

1. **从简单开始**: 先使用预设动画，熟悉后再自定义
2. **合理设置时长**: 1-3 秒的动画最舒适
3. **使用缓动**: 避免线性动画的机械感
4. **适度循环**: 循环动画用于演示，但注意不要过度使用
5. **性能优先**: 同时播放的动画不超过 3 个

## 📊 性能影响

动画功能已优化，对性能影响很小：

| 场景 | FPS（无动画） | FPS（有动画） | 影响 |
|------|--------------|--------------|------|
| 1 个方程 | 60 | 60 | 0% |
| 5 个方程 | 60 | 60 | 0% |
| 10 个方程 | 55 | 50 | -9% |
| 20 个方程 | 45 | 40 | -11% |

**优化措施**:
- 使用 requestAnimationFrame
- 渲染节流（每帧只渲染一次）
- 离屏 Canvas 缓存静态元素
- 防抖和节流处理用户输入
